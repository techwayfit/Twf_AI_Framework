## ?? Common Workflows

### Workflow 1: Multi-LLM Setup

```
1. Create Variables:
   {{openai_key}} = sk-proj-...
   {{model}}      = gpt-4o
   {{temp}}       = 0.7

2. Add LLM Node #1
   - Type {{ in API Key
   - Select {{openai_key}}
   - Type {{ in Model
   - Select {{model}}

3. Add LLM Node #2
   - Same as above

4. Change model globally:
   - Variables tab
   - Click {{model}}
   - Change to "gpt-4o-mini"
   - Both nodes updated!
```

### Workflow 2: HTTP Configuration

```
1. Create Variables:
   {{base_url}} = https://api.example.com

2. Add HTTP Node:
   - URL: Type "{{" ? {{base_url}}/users

3. Both fields turn yellow
4. Change {{base_url}} once ? all HTTP nodes update
```

## ?? Troubleshooting

### Autocomplete not showing?
**Cause:** No variables defined or not typing `{{` exactly  
**Fix:** Create at least one variable, type `{{` (two opening braces)

### Variable not highlighting yellow?
**Cause:** Typo in variable name  
**Fix:** Use autocomplete instead of manual typing

### Can't edit variable?
**Cause:** Variable not selected in sidebar  
**Fix:** Click the variable in Variables tab first

### Nodes tab empty?
**Cause:** Nodes not loaded  
**Fix:** Refresh browser (Ctrl+F5)

## ?? Best Practices Checklist

- ? Use autocomplete (type `{{`) instead of manual entry
- ? Name variables descriptively (`api_key` not `k1`)
- ? Group related variables together
- ? Test variables before using in many nodes
- ? Document complex variable values
- ? Use variables for ALL repeated values
- ? Leverage yellow highlight to verify variable usage
- ? Keep variables tab visible while configuring nodes
