## ?? Common Patterns

### Pattern 1: Configuration Variables

```
Variables (Global Config):
{{api_key}}
{{model}}
{{temperature}}
{{max_tokens}}

Used in: All LLM nodes
Benefit: Single source of truth
```

### Pattern 2: Environment URLs

```
Variables:
{{dev_api_url}} = https://dev-api.example.com
{{prod_api_url}} = https://api.example.com

Usage:
Development: Use {{dev_api_url}}
Production: Use {{prod_api_url}}
```

### Pattern 3: Template Reuse

```
Variables:
{{company_name}} = TechWay Solutions
{{support_email}} = support@techway.com
{{company_slogan}} = Innovation Simplified

Prompt:
"From {{company_name}} - {{company_slogan}}
Contact us: {{support_email}}"
```

### Pattern 4: A/B Testing

```
Variables:
{{model_a}} = gpt-4o
{{model_b}} = gpt-4o-mini
{{test_split}} = 50

Nodes:
Route A: Use {{model_a}}
Route B: Use {{model_b}}
Split: {{test_split}}%
```
