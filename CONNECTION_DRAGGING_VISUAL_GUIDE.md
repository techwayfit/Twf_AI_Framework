## ?? User Interactions

### Mouse Events

| Action | Result |
|--------|--------|
| **Hover connection line** | Line gets thicker, turns red |
| **Click connection line** | Select connection (shows properties) |
| **Hover green circle** | Circle enlarges, cursor: grab |
| **Hover red circle** | Circle enlarges, cursor: grab |
| **Mousedown on green** | Start dragging source endpoint |
| **Mousedown on red** | Start dragging target endpoint |
| **Drag to valid port** | Port enlarges, glows green |
| **Drag to invalid port** | Port stays gray, no glow |
| **Release on valid port** | Reconnect! |
| **Release elsewhere** | Cancel drag, revert |

## ?? Reconnection Logic

### Source Endpoint (Green Circle)

```
Can connect to:
- ? Any INPUT port
- ? OUTPUT ports (same type)
- ? Current target node (already connected)
- ? Same node (self-connection)

Example:
LLM Node (output) ?  Can drag to: HTTP Node (input) ?
     Cannot drag to: Another LLM Node (output) ?
```

### Target Endpoint (Red Circle)

```
Can connect to:
- ? Any OUTPUT port
- ? INPUT ports (same type)
- ? Current source node (already connected)
- ? Same node (self-connection)

Example:
HTTP Node (input) ? Can drag to: Another LLM Node (output) ?
    Cannot drag to: Another HTTP Node (input) ?
```

## ?? Best Practices

### ? Do This
- Use connection dragging to reorganize workflow
- Hover to see enlarged endpoints before dragging
- Watch for green glow = valid target
- Release mouse to complete reconnection

### ? Avoid This
- Don't connect node to itself
- Don't connect same port types (output?output)
- Don't try to connect to already-connected target

## ?? Summary

**Feature:** Draggable Connection Endpoints

**Visual Elements:**
- ?? Green circle = source (where connection starts)
- ?? Red circle = target (where connection ends)
- ?? Blue line = connection path
- ?? Purple dashed = dragging target
- ?? Orange dashed = dragging source

**How to Use:**
1. Hover over endpoint (circle enlarges)
2. Mouse down (cursor: grabbing)
3. Drag to new port (valid ports glow green)
4. Release mouse (connection updates!)

**Benefits:**
- ? No need to delete and recreate connections
- ? Fast workflow reorganization
- ? Visual feedback for valid targets
- ? Professional UX
