- [ ] Can edit text parameters (model name, API key)
- [ ] Can edit number parameters (temperature, max tokens)
- [ ] Can toggle boolean parameters (maintain history)
- [ ] Can select from dropdown (provider, HTTP method)
- [ ] Can edit JSON parameters (headers, field mappings)
- [ ] Can edit textarea parameters (prompts, templates)
- [ ] Required fields are enforced (marked with *)

- [ ] JSON fields validate JSON syntax
- [ ] Parameters persist after Save
- [ ] Parameters restore after reload
