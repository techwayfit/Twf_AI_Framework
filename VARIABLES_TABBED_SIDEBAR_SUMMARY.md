## ?? How to Use

### Create a Variable

1. Click **"Variables"** tab in sidebar
2. Click **"+ Add"** button
3. Properties panel shows form
4. Enter name: `api_key`
5. Enter value: `sk-proj-abc123`
6. Click **"Create Variable"**
7. Variable appears in list!

### Edit a Variable

1. Click **"Variables"** tab
2. Click the variable you want to edit
3. Properties panel shows variable details
4. Edit the value field
5. Changes save automatically!

### Use a Variable in a Node

**Method 1: Type Manually**
```
1. Select a node
2. In any text parameter, type: {{api_key}}
3. Field turns yellow (has-variables)
```

**Method 2: Autocomplete (Recommended!)**
```
1. Select a node
2. In any text parameter, type: {{
3. Dropdown appears with all variables
4. Use ?/? to navigate
5. Press Enter to insert
6. Field turns yellow automatically
```

### Delete a Variable

1. Click **"Variables"** tab
2. Click the variable to edit
3. Properties panel shows variable
4. Click **"Delete Variable"** button
5. Confirm deletion
6. Variable removed!
